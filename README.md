# ⚡ Big-O Analyzer

A modern AI-powered code complexity analyzer built with Streamlit and Gemini AI.

Analyze algorithms for:

- ⏱️ Time Complexity
- 🧠 Space Complexity
- 💡 Optimization Suggestions
- 🛡️ Edge Case Detection
- 🧪 Automatic Unit Test Generation

Designed with a clean modern developer-tool UI inspired by modern SaaS and AI platforms.

---

# ✨ Features

## 🔍 Automatic Language Detection
Paste code in any programming language and the AI automatically detects it.

## ⏱️ Complexity Analysis
Get detailed:
- Time Complexity
- Space Complexity
- Big-O explanations

## 💡 Optimization Suggestions
AI suggests more optimal approaches when possible.

## 🛡️ Edge Case Detection
Detect important corner cases that could break implementations.

## 🧪 Unit Test Generation
Generate robust unit tests using:
- PyTest
- JUnit
- Jest
- and other language-specific frameworks

## 📈 Complexity Visualization
Interactive complexity growth graph for:
- O(1)
- O(log N)
- O(N)
- O(N log N)
- O(N²)

## 🎨 Modern UI
- Minimal AI SaaS inspired design
- Animated gradient background
- Responsive layout
- Glassmorphism cards
- Smooth animations
